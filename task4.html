Task 4: Report Findings

Instructions

Now that you've gathered the data, it's time to compile your findings. A good report should include the following:

  • Introduction to the target and the social media platforms investigated.
  • Details of each account or profile found, including usernames, profile details, and any suspicious activity.
  • Metadata analysis, including photos and locations extracted from the images.
  • Conclusion, summarizing the most valuable findings.

Use a tool like Markdown or a simple text editor to format your findings. If you use Markdown, you can easily convert it to a PDF for sharing.

Don't forget to anonymize any personal information before sharing the report.
