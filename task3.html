Task 3: Extract Metadata from Photos

Instructions

Use tools like ExifTool to extract metadata from images shared on social media profiles. Metadata can reveal hidden information like:

  • Camera model
  • Location (GPS coordinates)
  • Date and time of capture
  • Software used to edit the image

Follow these steps:

  1. Download an image from the target's profile.
  2. Use ExifTool to extract metadata. For example: exiftool image.jpg
  3. Analyze the extracted data for useful information.
